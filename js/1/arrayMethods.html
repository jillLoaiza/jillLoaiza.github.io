Array length

The length property returns the number of elements in an array.


        const arr = [1,2,3]
        arr.length() // returns 3

Array pop()

removes the last element from an array and returns that element. This method changes the array. 


    const arr = [1,2,3]
    arr.pop() // returns 3
    console.log(arr = [1,2,3]) // logs [1,2]

Array push()

adds the specified elements to the end of an array and returns the new length of the array. 


    const arr = [1,2,3]
    arr.push(44) //returns 4 (the new length of arr)
    console.log(arr) //logs [1,2,3,44]

Array shift()

removes the first element from an array and returns that removed element. This method changes the array. 


    const arr = [1,2,3]
    arr.shift() // returns 1
    console.log(arr) // logs [2,3]

Array unshift()

adds the specified elements to the beginning of an array and returns the new length of the array. 


    const arr = [1,2,3]
    arr.unshift("abc") // returns 4
    console.log(arr) // logs ["abc", 1,2,3]
